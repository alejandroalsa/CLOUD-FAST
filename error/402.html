Error 402

El error 402 indica que la solicitud fue rechazada debido a que el pago requerido no ha sido procesado. Este código se utiliza a menudo en sitios de comercio electrónico para indicar que se necesita un pago antes de que se pueda acceder a un recurso o servicio.
